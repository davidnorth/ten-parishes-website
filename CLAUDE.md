# Ten Parishes Website

A festival website allowing visitors to explore events 
with an admin system to allow managing the events, their locations etc.

# Technologies

PHP zero dependencies
SQLite 
File system based routing, like a simplified Razor pages
All pages cached to static HTML
Simple password login for admin
Cloudinary for image storage and transformation
Anything non-specific to this site goes in '@framework'


# Rules
Ask before using any dependency at all. Ideally we'll use non but may need a lite ORM or CSV generator etc

Don't create any CSS and keep HTML as minimal as possible but still semantically correct.
